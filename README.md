# open-svpflow

An SVPFlow reimplementation.

## Demo

[smoothie.z1x.us](https://smoothie.z1x.us)

[![open-svpflow running in the browser](docs/assets/demo.png)](https://smoothie.z1x.us)

This is a proof of concept using [WebCodecs](https://developer.mozilla.org/en-US/docs/Web/API/WebCodecs_API) for video decoding and encoding. It recreates the functionality of [couleur-tweak-tips/smoothie-rs](https://github.com/couleur-tweak-tips/smoothie-rs) in the browser.
